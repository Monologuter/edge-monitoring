我需要对接远程硬件设备摄像头推送的告警图片，为是HTTP的请求，响应体找一些数据。图片也是以字节的形式，需要BASE64反编译成二进制字节流，生成图片。你接收到的是字符串。

在硬件设备上会设置好服务器 也就是我的后端的地址 目前项目中也就是9000端口 然后ip也就是服务器的公网地址47.120.48.64  硬件请求的地址为后端的API接口地址

如下为API调用说明：

```java
API 调用说明 
1、 日期格式均采用：yyyy-MM-dd HH:mm:ss 
2、 接口均采用 HTTP 协议，采用最基本的 POST 方法，Content-Type 为 application/json 格式，返回数据结果 
3、 接口中的URL地址为本地地址，网络传输后是无法访问的；通过/device/image接口可以获取URL对应的二进制图片内容。 第三方用户需要显示图像的话需要自行保存和替换图片路径。
```

完成我上述需求  相关数据请进行入库操作 接收到的数据base64需要转成图片 并将图片存储在ftp文件服务器上 对应地址如下 http://47.120.48.64:9004/
